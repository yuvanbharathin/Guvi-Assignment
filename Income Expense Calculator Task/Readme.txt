# Income Expense Calculator

## Overview
The Income Expense Calculator is a web application that helps users track their income and expenses. It allows users to add, edit, and delete entries, 
providing an overview of their financial status. The application implements CRUD (Create, Read, Update, Delete) operations to manage data and uses local storage to persist data 
across sessions.
